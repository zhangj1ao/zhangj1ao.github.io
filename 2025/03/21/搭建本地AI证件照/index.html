AI证件照

HivisionIDPhoto是一款高效且轻量级的AI证件照制作工具，专为满足用户多样化需求而设计。它采用先进的人工智能模型，并结合完善的工作流程，实现了对不同用户拍照场景的智能识别、背景抠图和证件照的快速生成。无论是身份证、护照、签证，还是其他标准证件照，HivisionIDPhoto 都支持多种尺寸的制作，确保符合各类官方规格。非常适合适合个人及企业的灵活应用场景。

源码部署

需要提前部署Python环境。
Plain Text
复制
标题
git clone https://github.com/Zeyi-Lin/HivisionIDPhotos.git cd  HivisionIDPhotos pip install -r requirements.txt python app.py
​
需要主要的是，官方文档提到要下载权重文件hivision_modnet.onnx。我没下载，反正又不是不能跑。
Docker部署
拉取镜像文件
Plain Text
复制
标题
docker pull linzeyi/hivision_idphotos:v1 docker run -p 7860:7860 hivision_idphotos
​
目前，项目至此已支持美颜功能，后期还会新增换装。如果你觉得安装上面环境部署比较麻烦，还可以安装Windows版本的工具。另外在衍生项目中，已经支持小程序。大家可以移步作者项目中看看。
